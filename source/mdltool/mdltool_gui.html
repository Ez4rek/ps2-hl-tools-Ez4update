Operação do batch
Referência: resize de textura
🔲
Tile resize original Preenche a nova dimensão repetindo o conteúdo da textura original como um tile. Se a dimensão original não é múltipla da nova potência de 2 (ex: 100px → 128px), a repetição cria uma borda/costura visível que pode causar artefatos visuais no PS2.
⬛
Scale resize novo Estica a textura proporcionalmente para a nova dimensão usando nearest-neighbour com aritmética inteira. Não cria bordas artificiais — a textura simplesmente é esticada. Recomendado para modelos com texturas de tamanho arbitrário.
⚠ Se possível, pré-redimensione as texturas para potências de 2 (8, 16, 32, 64, 128, 256...) num editor de imagens antes de converter, para evitar qualquer tipo de artefato. Ambos os modos de resize são apenas fallback.
